# Active Superpowers Plans

This directory is only for active implementation plans that should guide new
work. Historical plans with unchecked task boxes have been moved to
`docs/superpowers/archive/plans/` so they remain available for archaeology
without looking like current backlog.

Before creating a new plan, read the current code and architecture docs first:

- `docs/architecture/overview.md`
- `docs/architecture/rules.md`
- the relevant `docs/architecture/*` topic file

Treat archived plans as historical context only. If an archived plan conflicts
with current code, committed contracts, or architecture docs, the current code
and architecture docs win.
